# Robot Controls Summary

## Driving (Swerve)
- **Drive Translation (Forward/Back):** Left Stick Y
- **Drive Translation (Left/Right):** Left Stick X
- **Drive Rotation:** Right Stick X
- **Slow Mode (30% Speed):** Right Bumper (Hold)
- **Auto-Aim / Lock to Target:** B Button (Hold)
- **Invert Translation (Flip Front/Back):** Left Bumper (Toggle)
- **Invert Turn (Flip Left/Right Rotation):** Left Trigger (Toggle)
- **Toggle Slew Rate Limiter:** Back / View Button (Toggle)
- **Zero Gyro (Reset Heading):** Right Trigger (Press)

## Intake & Shooter
- **Shoot (47 RPM):** Y Button (Toggle On/Off)
- **Run Intake (5V):** X Button (Toggle On/Off)
- **Reverse Intake & Agitator:** A Button (Hold)
- **Intake Pivot Up:** D-Pad Up (Hold)
- **Intake Pivot Down:** D-Pad Down (Hold)

*Note: The operator controller mirrors most of the driver's inputs. Field centric toggling and agitator toggling controls are currently available via SmartDashboard.*
